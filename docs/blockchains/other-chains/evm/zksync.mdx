## Overview

[zkSync](https://zksync.io/) is a Layer 2 scaling solution for Ethereum, leveraging zk-rollup technology to offer fast and low-cost transactions while maintaining a high level of security.

**As zkSync is EVM compatible**, you can follow the [Ethereum](/docs/blockchains/featured-chains/ethereum/javascript) documentation to send your first transaction and utilize all other wallet features.

## Configure zkSync

Ensure you have installed the Magic SDK and have access to your API key, follow the [quickstart](/docs/home/quickstart/cli) to get started.

<Tabs>
    <Tab title="Testnet">
```js
// Setting network to point to zkSync testnet
const magic = new Magic('YOUR_PUBLISHABLE_API_KEY', {
  network: {
    rpcUrl: 'https://sepolia.era.zksync.dev',
    chainId: 300,
  },
});

````
    </Tab>
    <Tab title="Mainnet">
```js

// Setting network to point to zkSync mainnet
const magic = new Magic('YOUR_PUBLISHABLE_API_KEY', {
  network: {
    rpcUrl: 'https://mainnet.era.zksync.io',
    chainId: 324,
  },
});
````

    </Tab>

</Tabs>
