Magic's SDKs integrate into any web or mobile application to enable:

* Passwordless [authentication](/docs/authentication/overview) (e.g., email OTP, magic link, social, WebAuthn, 3rd party ID provider)

* Non-custodial [wallets-as-a-service](/docs/wallets/customization/widget-ui) with best-in-class security and performance

* Customizable embedded wallet [widget UI](/docs/wallets/customization/widget-ui)

* Compatibility with [30+ blockchains](/docs/blockchains/overview)

## Wallets
Magic creates wallets for your users when they sign up. Our patented [Delegated Key Management System (DKMS)](/docs/home/security/product-security#private-keys-can-be-lost-or-stolen-how-do-you-protect-my-private-key) keeps keys secure and non-custodial.
