## API Wallets

**Server-side wallet management with enterprise-grade security**

- **Core API**: Maximum control with AWS Nitro TEE security and advanced key sharding
- **Express API**: Streamlined integration with JWT authentication and trusted execution environment technology
- **Non-custodial security**: Users maintain full control over their private keys
- **Multi-chain support**: EVM chains, Solana, and Bitcoin
- **Production-ready**: Enterprise-grade infrastructure for scalable applications

## Embedded Wallets

**Seamless Web3 onboarding with non-custodial wallets**

- **Passwordless authentication**: Email OTP, SMS, social logins, and more
- **Instant wallet creation**: Users get wallets immediately upon authentication
- **Patented DKMS**: Delegated Key Management System for secure key management
- **Multi-chain support**: 30+ blockchain networks
- **Customizable UI**: Embedded wallet widgets with full branding control
