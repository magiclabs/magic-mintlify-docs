## Overview

[Berachain](https://www.berachain.com/) is an EVM-compatible L1 blockchain built on-top of the Cosmos-SDK, which uses [Proof-of-Liquidity](https://docs.berachain.com/learn/#proof-of-liquidity-overview) consensus.

**As Berachain is EVM compatible**, you can follow the [Ethereum](/docs/embedded-wallet/blockchains/ethereum/javascript) documentation to send your first transaction and utilize all other wallet features.

## Configure Berachain

Ensure you have installed the Magic SDK and have access to your API key, follow the [quickstart](/docs/embedded-wallet/quickstart/overview) to get started.

```js Testnet icon="square-js"
// Setting network to point to Berachain Artio testnet
const magic = new Magic("YOUR_PUBLISHABLE_API_KEY", {
  network: {
    rpcUrl: "https://artio.rpc.berachain.com/",
    chainId: 80085,
  },
});
```

## Compatibility

- All `Auth`, `User` and `Wallet` module methods
- All EVM Provider functionality to respond to supported [RPC methods](/docs/embedded-wallet/api/client-side-sdks/web#evm-rpc-methods)
